## **Design Guidelines for All Slides:**

### Color Scheme:
- Primary: #1976D2 (Blue)
- Secondary: #9C27B0 (Purple)
- Accent: #4CAF50 (Green)
- Background: White/Light Gray (#F5F5F5)
- Text: Dark Gray (#333333)

### Typography:
- Headings: Montserrat Bold (32-40pt)
- Subheadings: Montserrat SemiBold (24-28pt)
- Body Text: Open Sans Regular (16-18pt)
- Code/Technical: Consolas/Monaco (14pt)

### Visual Elements:
- Use consistent icons from Material Icons or Font Awesome
- Include screenshots from the actual application
- Use charts and graphs where applicable
- Maintain white space for clean look
- Add subtle animations (fade-in, slide-in) for engagement

### Layout:
- Consistent header with logo on each slide
- Footer with slide numbers
- Use 2-3 column layouts for content-heavy slides
- Balance text with visuals (60% visual, 40% text)
